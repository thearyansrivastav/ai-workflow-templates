# AI System Prompts

A structured library of system prompts for common AI workflow tasks.

These are working prompts — used in actual automation workflows,
tested against real tasks, and updated based on output quality.

This is not a collection of cute ChatGPT tricks.

---

## Structure

Prompts are organized by use case:

| Folder | Contents |
|---|---|
| `writing/` | Email drafting, content editing, documentation |
| `business/` | Lead qualification, proposals, research summaries |
| `operations/` | Task extraction, meeting notes, status updates |
| `data/` | Extraction, classification, entity recognition |
| `code/` | Code review, documentation generation |
| `meta/` | Prompts about prompts; format guide |
